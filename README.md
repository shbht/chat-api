# chat-api
Api component for chat application

## Setup

1. Check the npm packages:

    ```
    npm install
    ```

2. Create build:

    ```
    grunt
    ```

2. Start the application

    ```
    node dist/api.js
    ```

## Managing the project with Grunt

* Runs eslint, babel:dist

    ```
    grunt
    ```

* Compiles the .es6 files to .js

    ```
    grunt babel:dist
    ```
